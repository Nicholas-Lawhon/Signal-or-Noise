## Findings

- **[BLOCKER] Full 20-round runs cannot finish.** On Round 20, Lock In applies the round and advances `currentRoundIndex` to 20, but while `view === 'locked'` the page still reads `scenario = scenarios[run.currentRoundIndex]`; that is undefined and the guard renders `No scenario available.` before the locked/reveal/summary branches. This fails H001-7 and the H004 full-run walkthrough.
- **[BLOCKER] Selected confidence color ramp does not render.** The UI builds Tailwind classes dynamically (`border-${color}`, `bg-${color}/10`, `text-${color}`), so the selected cyan/green/amber/violet styles are absent from generated CSS. Computed selected Low/Medium/High/All-In borders stayed default gray and backgrounds stayed transparent. This fails H002-5.
- **[MAJOR] Hidden Amazon card is overly identifying.** The hidden title `Everything Store, Everything Crash` and clue phrase `world's most customer-centric company` are effectively Amazon-identifying, conflicting with `soul.md` content-integrity posture against unmistakable identifying hidden-card content. This was prescribed by H001 sample data, so treat as orchestrator/content cleanup rather than an implementor-only miss.
- **[MINOR] Netflix precision mismatch remains.** Netflix data uses `actualReturnPercent: 11.36`, so the app displays `+1136.0%`; H004 expected this known item to be recorded and recommends correcting the sample value to `11.356` in a future data pass.
- **[MINOR] Dev server port note.** `pnpm dev` did serve successfully, but port 3000 was not available in this session and Next selected `http://localhost:3001`; `localhost:3000` returned connection failed while 3001 returned 200.

## Locked-Rule Spot Checks

- **Scoring values:** `packages/game-engine/src/confidence.ts` matches Low 10%/+1, Medium 40%/+2, High 70%/+3, All-In 100%/+5; Pass is `-0.25` in `scoreRound`.
- **D014/D015/D016:** Source and browser checks confirm wrong All-In busts to $0, guess bonus/penalty affects only Signal Score, and bankruptcy floor is `BANKRUPTCY_FLOOR = 1`.
- **Decimal returns / zero-return rule:** `actualReturnPercent` values are decimals in `sampleScenarios.ts`; `scoreRound` uses `rawReturn > 0`, so exactly zero is incorrect per D007.

- **Anti-leakage display:** Pre-lock round text did not show literal company name, ticker, actual return, or outcome chart. Local prototype still contains data client-side, which is allowed for Phases 0-2.
