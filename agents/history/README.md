# Agent History Archives

Archived state lives here so active agents do not load closed-phase material by
default. A handoff must name an exact archive path or section before opening it.
Each phase bundle keeps its historical handoffs, reports, audits, consultations,
and retired machine artifacts together; active `agents/` folders contain only
current/next work and artifacts a live handoff needs.

| Path | Contents |
|---|---|
| `progress_phase_0_3.md` | Phase 0-3 summaries and detailed archived session log |
| `phase_0_3/` | Closed Phase 0-3 handoffs, reports, audits, and consultations |
| `progress_phase_4a.md` | Phase 4 Part A session log |
| `phase_4a/` | Closed Phase 4A handoffs, reports, consultations, and retired Gate 2 exports |
| `phase_4b_legacy/` | Retired Phase 4B H033-H037/R046-R063 workflow, Gate 2 artifacts, C004 consultation, session log, templates, and diff-summary tool |
| `decisions_phase_0_4.md` | Full decision bodies; use only when a named decision needs detail |
