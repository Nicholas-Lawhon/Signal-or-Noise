## Summary

Scaffolded the full pnpm monorepo with Next.js 14 web app, pure TypeScript game engine (`packages/game-engine`), and a placeholder content package. Implemented all game scoring math (confidence levels, long/short/pass, bankruptcy, signal score) with 18 passing Vitest tests. Built a complete playable Classic Run prototype: landing page → mode select → difficulty pick → 20-round scenario loop with Long/Short/Pass + confidence buttons, reveal screen, and end-of-run summary. Six hardcoded prototype scenarios provide content. All scoring lives in `packages/game-engine`; the web app imports it.
